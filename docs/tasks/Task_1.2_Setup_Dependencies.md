# Task: [Setup - Dependencies]

**Phase**: Phase 1 - Setup / Foundation

## Description
Add the dependencies for Hilt, Jetpack Compose, Room, Retrofit, OkHttp, Kotlin serialization, and KSP in the project-level and app-level `build.gradle` files.

## Acceptance Criteria
- [ ] The project syncs successfully with Gradle without errors.
- [ ] The library versions are compatible with each other.
- [ ] Annotation processing uses KSP instead of kapt.
- [ ] JSON serialization is configured with Kotlin serialization instead of Gson or Moshi.
